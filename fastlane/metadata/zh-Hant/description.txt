CampNet 是你的校園網助手。它能在後台自動登錄校園網，免去你頻頻打開網頁登錄的煩惱。

主要功能：

====== 自動管理 ======
• 自動登錄
• 自動下線過期鏈接（防蹭網）

====== 流量監視 ======
• 直觀查看流量趨勢
• 當月網費估計
• 在流量不足時發出警報

====== 高級功能 ======
• 在線設備管理
• 多賬戶管理
• 上線其他 IP

====== 通用框架 ======
• 使用配置文件對校園網進行適配
• 當前支持：
    • 清華大學
• 沒有找到你的學校？在 https://github.com/ClumsyLee/CampNet-Configurations 提交 issue 或 pull request ！
