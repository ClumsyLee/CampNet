CampNet 是你的校园网助手。它能在后台自动登录校园网，免去你频频打开网页登录的烦恼。

主要功能：

====== 自动管理 ======
• 自动登录
• 自动下线过期链接（防蹭网）

====== 流量监视 ======
• 直观查看流量趋势
• 当月网费估计
• 在流量不足时发出警报

====== 高级功能 ======
• 在线设备管理
• 多账户管理
• 上线其他 IP

====== 通用框架 ======
• 使用配置文件对校园网进行适配
• 当前支持：
    • 清华大学
• 没有找到你的学校？在 https://github.com/ThomasLee969/CampNet-Configurations 提交 issue 或 pull request ！
